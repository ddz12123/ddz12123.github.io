# 图标组件库

:::tip 图标选择指南
选择合适的图标库可以提升应用的视觉效果和用户体验。以下是一些常用的图标组件库推荐。
:::

## 菜鸟图标

:::info 菜鸟图标特点
菜鸟图标提供丰富的矢量图标资源，适合各种 Web 项目使用。
:::

| 图标库 | 特点 | 官网链接 |
|--------|------|---------|
| **菜鸟图标** | 提供大量免费矢量图标，支持多种格式下载 | [官网](https://icon.sucai999.com/) |

## IconPark

:::note IconPark 优势
IconPark 是字节跳动开源的图标库，提供丰富的图标资源和强大的定制功能。
:::

| 图标库 | 特点 | 官网链接 |
|--------|------|---------|
| **IconPark** | 支持在线编辑、多格式导出、主题定制等功能 | [官网](https://iconpark.oceanengine.com/) |

## 其他推荐图标库

:::success 更多选择
以下是一些其他优秀的图标库，可以根据项目需求选择使用：
:::

| 图标库 | 特点 | 官网链接 |
|--------|------|---------|
| **Iconify** | 集成多个图标库，提供统一的图标接口 | [官网](https://iconify.design/) |
| **Feather Icons** | 简洁美观的开源图标集 | [官网](https://feathericons.com/) |
| **Heroicons** | 精美的 SVG 图标集，由 Tailwind CSS 团队开发 | [官网](https://heroicons.com/) |
| **Lucide** | 开源图标库，提供数百个精美图标 | [官网](https://lucide.dev/) |

## 使用建议

:::warning 最佳实践
1. 根据项目风格选择合适的图标库
2. 考虑图标的可访问性和语义化
3. 优化图标加载性能，考虑使用 SVG 格式
4. 保持图标风格的一致性
:::
