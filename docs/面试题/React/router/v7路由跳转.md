# react-router v7 路由跳转

可以使用以下方式：`<Link>` `<NavLink>` `useNavigate`

```tsx
import { NavLink } from "react-router";

<NavLink to="/" end>

<Link to="/login">Login again</Link>    
```

useNavigate:

```tsx
import { useNavigate } from "react-router";
let navigate = useNavigate();

navigate("/dashboard");
```
